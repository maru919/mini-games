# mini-games
A collection of mini games built for fun and experimentation.

## Games

| ゲーム | 説明 | 起動 |
|--------|------|------|
| [神経衰弱](./memory/) | 2人対戦の神経衰弱。絵文字・単色・EBiDAN の3種のカードセット対応 | `cd memory && npm run dev` |
| [テトリス](./tetris/) | 1人用テトリス。7-bag方式・レベル制・スコアシステム | `cd tetris && npm run dev` |
